Project Description

This project applies different Machine Learning techniques on the Customer Personality Analysis dataset.

The project includes:

Data Preprocessing
Feature Engineering
Regression Models
Classification Models
Clustering Analysis

Dataset

Customer Personality Analysis Dataset (marketing_campaign.csv)

Data Preprocessing

The following preprocessing steps were applied:

Missing values handling using Median Imputation
Invalid data filtering
Feature Engineering
Label Encoding
One-Hot Encoding
Feature Scaling using StandardScaler

Regression Models

The following regression models were used to predict TotalSpending:

Linear Regression
Ridge Regression
Decision Tree Regressor

Evaluation Metrics:

MSE
RMSE
R² Score

Classification Models

The following classification models were used to predict customer Response:

Logistic Regression
K-Nearest Neighbors (KNN)
Random Forest

Evaluation Metrics:

Accuracy
Precision
Recall
F1 Score
Confusion Matrix

Clustering

K-Means clustering was used to segment customers based on:

Income
Age
TotalSpending
TotalChildren

Additional methods:

Elbow Method
PCA Visualization
